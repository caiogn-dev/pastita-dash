# Navbar / IA Reorganization Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Reorganize the dashboard top navbar into a clear B2B mental model (operação · crescimento · config da loja), move account-level items to an avatar menu, and gate the Automação section behind progressive disclosure — without rewriting any page or changing any route.

**Architecture:** Today `Navbar.tsx` builds its `sections: NavSection[]` array inline inside the component. We extract that into a pure, unit-testable builder (`buildNavSections`) so the new information architecture can be tested without rendering. A small hook (`useAutomationEnabled`) computes the progressive-disclosure predicate (WhatsApp connected AND ≥1 agent) non-blockingly and feeds it into the builder. The avatar block (top-right) becomes a dropdown reusing the existing portal pattern; account-level links and logout move into it. The mobile drawer renders the same builder output plus an account block.

**Tech Stack:** React + TypeScript, react-router-dom (`NavLink`), Zustand stores, `@tanstack/react-query` v5, Heroicons, Jest (`jest.config.cjs`, jsdom).

## Global Constraints

- **No page rewrites, no route changes.** Only navigation menu structure changes. Every link keeps pointing at its current route.
- **No new dependencies.** Reuse the existing portal dropdown mechanism (`PortalDropdown`) — do not add a menu/popover library.
- **Navbar is `src/components/layout/Navbar.tsx`** (top bar with portal dropdowns). `Sidebar.tsx` is dead legacy — do not touch it (per CLAUDE.md).
- **`storeHref(path)`** = `/stores/${store.slug||store.id}/${path}`, or `/stores` when no store selected. Preserve exactly.
- **Gating must never block navbar render.** While the agents signal loads or if it errors, treat automation as disabled (`false`).
- **No tenant slug hardcoding** as runtime fallback (per CLAUDE.md).
- **TS/build must stay green** — a TS error blocks the Vercel deploy. `npx tsc --noEmit` and `npm run build` must pass before final commit.
- **Baseline test state:** pre-existing failures in `test_ComboForm` (combos.test.tsx) are known; do not let them mask new regressions, but they are not introduced by this work.

## Task 1: `useAutomationEnabled` gating hook

**Files:**
- Create: `src/hooks/useAutomationEnabled.ts`
- Test: `src/hooks/useAutomationEnabled.test.tsx`

**Interfaces:**
- Consumes: `useStore()` → `{ store }` where `store: Store | null` and `Store` has `whatsapp_number: string` and `integrations_count: number` (`src/services/storesApi.ts:60,80`); `agentsService.getAgents(): Promise<Agent[]>` (`src/services/agents.ts:236`, default export).
- Produces: `useAutomationEnabled(): boolean` — `true` only when the store has a WhatsApp signal AND ≥1 agent; `false` while loading or on error.

**Predicate:**
```
hasWhatsApp = Boolean(store?.whatsapp_number) || (store?.integrations_count ?? 0) > 0
hasAgent    = (agents?.length ?? 0) > 0
enabled     = hasWhatsApp && hasAgent
```
Agents are fetched via `useQuery` with a high `staleTime` and `enabled: hasWhatsApp` (no point fetching agents if there's no WhatsApp yet). On error or while pending, `agents` is `undefined` → `hasAgent` is `false` → hook returns `false`.

```ts
// src/hooks/useAutomationEnabled.ts
import { useQuery } from '@tanstack/react-query';
import { useStore } from './useStore';
import agentsService from '../services/agents';

/**
 * Progressive disclosure for the "Automação" nav section.
 * Returns true only when the current store has a WhatsApp signal AND >=1 agent.
 * Never throws and never blocks render: while loading or on error it returns false.
 */
export function useAutomationEnabled(): boolean {
  const { store } = useStore();

  const hasWhatsApp = Boolean(store?.whatsapp_number) || (store?.integrations_count ?? 0) > 0;

  const { data: agents } = useQuery({
    queryKey: ['agents', 'gating'],
    queryFn: () => agentsService.getAgents(),
    enabled: hasWhatsApp,
    staleTime: 5 * 60 * 1000,
    retry: false,
  });

  const hasAgent = (agents?.length ?? 0) > 0;
  return hasWhatsApp && hasAgent;
}
```

```tsx
// src/components/layout/AccountMenu.tsx
import React, { useState, useRef, useEffect, useCallback } from 'react';
import { createPortal } from 'react-dom';
import { NavLink, useNavigate } from 'react-router-dom';
import {
  ArrowRightOnRectangleIcon, BuildingStorefrontIcon, DevicePhoneMobileIcon,
  LinkIcon, Cog6ToothIcon, SparklesIcon,
} from '@heroicons/react/24/outline';
import { useAuthStore } from '../../stores/authStore';
import { useStore } from '../../hooks/useStore';

interface AccountLink {
  name: string;
  href: string;
  icon: React.ComponentType<{ className?: string }>;
  sectionHeader?: string;
}

const ACCOUNT_LINKS: AccountLink[] = [
  { name: 'Todas as Lojas',  href: '/stores',      icon: BuildingStorefrontIcon },
  { name: 'Contas WhatsApp', href: '/accounts',    icon: DevicePhoneMobileIcon, sectionHeader: 'Integrações' },
  { name: 'Conexões',        href: '/connections', icon: LinkIcon },
  { name: 'Preferências',    href: '/settings',    icon: Cog6ToothIcon, sectionHeader: 'Conta' },
  { name: 'Plano',           href: '/plano',       icon: SparklesIcon },
];

export const AccountMenu: React.FC = () => {
  const { user, logout } = useAuthStore();
  const { store } = useStore();
  const navigate = useNavigate();
  const [open, setOpen] = useState(false);
  const [pos, setPos] = useState({ top: 0, right: 0 });
  const btnRef = useRef<HTMLButtonElement>(null!);

  const close = useCallback(() => setOpen(false), []);

  useEffect(() => {
    if (!open) return;
    const update = () => {
      if (!btnRef.current) return;
      const r = btnRef.current.getBoundingClientRect();
      setPos({ top: r.bottom + 6, right: window.innerWidth - r.right });
    };
    update();
    const onDoc = (e: MouseEvent) => {
      if (btnRef.current?.contains(e.target as Node)) return;
      if (document.getElementById('account-menu-portal')?.contains(e.target as Node)) return;
      close();
    };
    window.addEventListener('scroll', update, true);
    window.addEventListener('resize', update);
    document.addEventListener('mousedown', onDoc);
    return () => {
      window.removeEventListener('scroll', update, true);
      window.removeEventListener('resize', update);
      document.removeEventListener('mousedown', onDoc);
    };
  }, [open, close]);

  const initial = user?.first_name?.[0] || user?.username?.[0] || 'U';

  return (
    <div className="flex items-center pl-2 border-l border-white/20">
      <button
        ref={btnRef}
        onClick={() => setOpen((v) => !v)}
        aria-label="Menu da conta"
        className="flex items-center gap-1.5 rounded-md p-0.5 hover:bg-white/10 transition-colors"
      >
        <div className="w-6 h-6 bg-white/15 rounded-full flex items-center justify-center flex-shrink-0">
          <span className="text-[10px] font-semibold text-white">{initial}</span>
        </div>
        <span className="block max-md:hidden text-xs font-medium text-white truncate max-w-[70px]">
          {user?.first_name || user?.username}
        </span>
      </button>

      {open && createPortal(
        <div
          id="account-menu-portal"
          style={{ position: 'fixed', top: pos.top, right: pos.right, zIndex: 9999 }}
          className="w-56 bg-surface border border-border-token rounded-xl shadow-2xl py-1"
        >
          <p className="px-3 pt-2 pb-1 text-[11px] font-semibold text-fg-token truncate">
            {store?.name || 'Cardapidex'}
          </p>
          {ACCOUNT_LINKS.map((l) => (
            <React.Fragment key={l.href}>
              {l.sectionHeader && (
                <p className="px-3 pt-2.5 pb-1 text-[10px] font-semibold uppercase tracking-wider text-fg-muted-token border-t border-border-token mt-1">
                  {l.sectionHeader}
                </p>
              )}
              <NavLink
                to={l.href}
                onClick={close}
                className={({ isActive }) =>
                  `flex items-center gap-2.5 px-3 py-2 text-sm transition-colors ${
                    isActive ? 'bg-brand text-[var(--brand-strong)] font-medium' : 'text-fg-token hover:bg-surface-2'
                  }`
                }
              >
                <l.icon className="w-4 h-4 flex-shrink-0" />
                <span>{l.name}</span>
              </NavLink>
            </React.Fragment>
          ))}
```
